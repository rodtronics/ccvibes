# lore/ - Game Lore and Story

Narrative content and world-building documents. These inform modal content, activity flavor text, and the overall game tone.

## Files

| File | Purpose |
|------|---------|
| `07_lore_philosophy.md` | Lore design philosophy - tone, themes, narrative approach. Replaces deleted `00_lore_philosophy.md`. |
| `lore_global_meta.md` | Global meta-narrative framework - overarching story structure, world rules, progression arcs. |
| `lore_primordial_notes.md` | Notes and ideas for the Primordial branch storyline. |
| `lore_primordial_story.md` | Full Primordial branch narrative - story beats, character arcs, dialog drafts. |

## Related
- `gemini_lore/` (sibling folder) contains alternate lore drafts generated by Gemini.
- Modal content in `data/modals.json` is the in-game delivery format for lore (intro, tutorials, story reveals).
- Activity descriptions in `data/activities.json` carry per-job flavor text.
