How to Use

  • Paste your ASCII art into the left panel
  • Click "Convert to JSON" to generate the JSON string
  • The output escapes newlines and quotes properly for JSON
  • Click "Copy to Clipboard" to copy the JSON string
  • Paste the string into your modal content field in modals.json

Markup Support

  • Bold: **text**
  • Dim: ~~text~~
  • Colored text: {{color}}text{{/}}
  • Background: {{bg:color}}text{{/}}
  • Line breaks: Use actual line breaks in your ASCII
