# Crime Committer VI - New GPT Attempt

This folder is a clean-room implementation based only on the founding documents.

## Run

Serve the folder so the JSON data loads correctly:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000/new_gpt_attempt/`.

## Controls

- Tabs: J / A / C / S
- Jobs list: Arrow keys, Enter, 1-9
- Options: Arrow keys, Enter, 1-9, Backspace
- Repeat: + / - (countdown), R (toggle infinite)
- Active runs: Arrow keys, X to stop, R to stop repeat
- Crew modal: Arrow keys, Left/Right to cycle, Enter to confirm, Esc to cancel
- Settings: G (glow), T (gradients), + / - (font size)
